## Inisialisasi NPM
```bash
npm init -y
```

## Install Tailwind , PostCSS , Dan Autoprefixer

```bash
npm i -D tailwindcss postcss autoprefixer
# or
npm i -D tailwindcss
```

## Inisialisasi Tailwind CSS
```bash
npx tailwindcss init
```
## Compile 
```bash
npx tailwindcss -i input_file_css -o output_file_css watch
```
